Firebase

El proveedor más cercano a Angular.

Firebase es el proveedor de BaaS favorito para cualquier aplicación de Angular.

AngularFire

Este es el nombre que recibe el módulo que nos permite enlazar cualquier aplicación de Angular con nuestra cuenta de Firebase de manera fácil y rápida. Lo primero que debemos hacer es crear una cuenta en Firebase.com. Una vez hecho eso, debemos enlazar el módulo con estos scripts:

			
<!-- Firebase -->
<script src="https://cdn.firebase.com/js/client/2.2.4/firebase.js"></script>
<!-- AngularFire -->
<script src="https://cdn.firebase.com/libs/angularfire/1.1.1/angularfire.min.js"></script>

Luego inyectamos el módulo de AngularFire a nuestra aplicación:

			
var app = angular.module("app", [
	
	"ngRoute",
	"firebase"

]);

Esto nos permitirá agregar los distintos servicios que presta AngularFire para nuestro desarrollo. son 3: $firebaseObject, $firebaseArray y $firebaseAuth. Pueden leer más de que trata cada uno en la documentación de AngularFire.

El siguiente paso es inyectar los servicios a nuestros controladores:

			
app.controller("TodoController", function($scope, $firebaseArray) {
  var ref = new Firebase("https://<SU-APP-EN-FIREBASE>.firebaseio.com/todos");
});

En este punto ya tienen su aplicación conectada con Firebase.
