Javascript - Funciones

Bloques de código reutilizables

Sería muy tedioso tener que escribir el mismo código miles de veces. Es por esto que existen las funciones y en Javascript, funcionan un poco distinto de lo que estamos acostumbrados.

Qué es una Función ?

Una función es un bloque de código que puede ser reutilizado en cualquier parte del documento. En su forma más básica, una función tiene la siguiente estructura:

				
function miFuncion() {
	console.log("Hello World");
}

Esta función se podría usar en cualquier parte del documento de Javascript que estamos escribiendo, tan solo al llamar "miFuncion()".

Las funciones traen consigo un tema importante que es el alcance (scope). Variables definidas dentro de una función, solo pueden ser accedidas dentro de esa misma función. Ejemplo:

				
function foo() {
	var x = 16;
}

x != y // x no existe y esto botará un error (x is not defined o algo así)
Nota: lo mismo pasa dentro de repetitivas y condicionales, también tienen un alcance definido.

Tipos de Funciones

Existen distintos tipos de funciones. En total son 3:

  • Funciones Declaradas
  • Funciones Expresadas
  • Funciones Autoinvocadas
En realidad, solo existen las 2 primeras, pero la tercera es interesante y vale la pena pensarla como un tipo de función.

Ejemplos de los 3 tipos son:

				
// Función Declarada, se invoca llamando "bar()".

function bar() {
	console.log("asdf");
}

// Función Expresada, se invoca llamando x()

var x = function() {
	console.log("x");
};

// Funciones autoinvocadas, se ejecutan automáticamente

(function () {
    var x = "Hola!!";
})();
Es importante saber que las funciones en javascript son llevadas automáticamente al inicio del documento, por lo que se pueden llamar antes de ser declaradas / expresadas.

Parámetros y Retorno

Las funciones no serían muy útiles si no pudieran tomar algo y volverlo otra cosa. Eso es exactamente lo que hacen los parámetros y el retorno. Una función puede tomar n cantidad de elementos y usarlos dentro suyo para transformarlos y devolver otra cosa.

				
var suma = function(a, b) {
	return a + b;
};

var resta = function(c, d) {
	return c - d;	
};

var multiplicacion = function(e, f) {
	return e * f;
}

// Cuál es el valor de x ?

var x = multiplicacion(suma(2, 2), resta(5,3));
Las funciones también son variables y pueden ser usadas como parámetros para otras funciones !

Callbacks

En Javascript es muy común el concepto de "Callbacks". Un callback es una función que se ejecuta al terminar de realizarse otra función o, que se ejecuta en el momento en que se cumple alguna condición. Los siguientes son 2 ejemplos que se ven mucho en el desarrollo web en cualquier cosa que use Jquery:

				
$("document").ready(function(){
	// Realizar todas las acciones de mi código luego de que se cargue el DOM.
});

$(".foo").click(function(){
	// Cuando se clickee algo con la clase "foo", realizar una acción.
});
