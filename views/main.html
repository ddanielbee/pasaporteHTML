Pasaporte HTML 5

Aprendiendo de la Web moderna paso a paso.

Aunque el nombre del pasaporte sea HTML 5, realmente lo que vamos a aprender es lo que es la web moderna. No van a ser expertos luego de estas 16 horas, pero sabrán mucho más que antes y dónde aprender aun más.

Lo que se va a aprender

Nota: Este curso supone que las personas que lo toman tienen una experiencia básica con HTML y CSS y al menos un acercamiento inicial a Javascript. Por esta razón no se va a profundizar en temas demasiado específicos de cada uno.

Lo que vamos a hacer

Una aplicación de ToDo's, creada con Bootstrap, AngularJS y usando Firebase como Backend.

Día 1

El primer día, vamos a escribir el esqueleto de la aplicación. Usaremos Bootstrap como base para el CSS.

Día 3

El tercer día, luego de aprender Angular, vamos a llevar nuestra aplicación a un punto en el que funcione de manera local.

Día 4

Para el final del cuarto día, vamos a haber implementado Firebase, para tener una aplicación que funcione fuera del ambiente local y tenga persistencia de datos.
