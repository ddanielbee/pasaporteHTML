Ejercicio Teórico CSS

Entender lo que los demás hacen es el camino para el propio aprendizaje.

Buscar referencias es una parte integral de cualquier proceso de diseño. En este ejercicio, vamos a ver una referencia y a tratar de entender lo que se hizo.

Busquen un website que les guste, cualquiera. Encuentren algún efecto que crean es CSS y traten de entender cómo se hizo.

Como parte del ejercicio, abran Elede Lab y trabajaremos con ese.

El siguiente es el ejercicio de observación a realizar en este caso:

  • 1. Cómo se logró el efecto de la navegación superior ?
  • 2. Cómo se logró el efecto de los botones en el home ?
