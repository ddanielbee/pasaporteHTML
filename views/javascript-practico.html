Ejercicio Práctico

Aprender a usar el código de otros es de vital importancia en Javascript. Y aprender a leer documentación de librerías también lo es.

Para este ejercicio, vamos a construir una galería de imágenes traídas de Instagram. Para esto, vamos a hacer uso de InstafeedJS y de Foundation.

Su misión es crear una galería de fotos de un tag específico, que vengan de Instagram. Debe ser responsive y todas las fotos deben ser del mismo tamaño.

Puntos de bonus para el que además de esto le ponga la descripción a la imagen.

Pasos a seguir:

  • 1. Encontrar el elemento de foundation que permita hacer este tipo de galerías.
  • 2. Leer la documentación de instafeed.js y seguir los pasos para llamar las imágenes con un tag de su elección
  • 3. Cambiar el template default que usa instafeed.js por el que necesita foundation para funcionar.
