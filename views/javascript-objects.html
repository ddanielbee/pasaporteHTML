Javascript - Objetos

Contenedores flexibles de información

Los objetos lo son todo en Javascript. Si alguien entiende los objetos de Javascript, entiende Javascript.

Qué son ?

En Javascript, todo es un objeto. Un entero es un objeto. Un string es un objeto. Una función es un objeto y un objeto es un objeto. Los objetos son variables. Entonces podemos tener objetos que tengan un solo valor, como:

				
	var x = "lalala";

Pero los objetos también pueden tener muchos valores, como:

				
	var x = {x: "5", y: "Hello World !", w: 26};

E incluso, los valores pueden ser de distintos tipos como se ve en el ejemplo anterior. Puede incluso pasar lo siguiente:

				
	var x = {
			x: "hola",
			y: "mundo",
			w: {
				x: "wow",
				y: "Inception"
			}
		};

Objetos dentro de objetos dentro de objetos dentro de objetos... y ya entienden la idea.

Notación

Para escribir un objeto en Javascript, se utiliza la notación con corchetes x = {}; Cada una de las propiedades lleva un nombre (sin comillas), seguido de dos puntos ":" y luego su valor. Las propiedades son separadas por comas "," excepto la última donde no es necesario. Para acceder a las propiedades de un objeto:

				
	var x = {
			x: "hola",
			y: "mundo",
			w: {
				x: "wow",
				y: "Inception"
			}
		};

	var y = x.x; 	// y = "hola";
	var y = x["x"]; // y = "hola";

Para redeclarar propiedades y/o definir nuevas, se hace de la misma forma: 

				
	var x = {
			x: "hola",
			y: "mundo",
			w: {
				x: "wow",
				y: "Inception"
			}
		};

	x.x = "ahora no soy hola";
	x["x"] = "ahora no soy hola";
	x.foo = "bar";

	/* Si mostráramos los contenidos de x ahora veríamos algo así:
	var x = {
			x: "hola",
			y: "mundo",
			w: {
				x:"wow",
				y:"Inception"
			},
			foo: "bar"
		};
	*/

Prototipos

Javascript está basado en Prototipos. La creación de prototipos como tal es algo un poco muy avanzado para este curso, pero el conocimiento de su existencia no le hace daño a nadie. Cada una de las variables que utilizamos tiene un prototipo dentro del core de Javascript. Piensen en un prototipo de Javascript como en una clase de Java. Tienen propiedades que pueden ser llamadas por cualquiera de los miembros de su clase. Es así como podemos hacer lo siguiente:

				
	var x = "soy un string";
	var y = x.length();

	// y es el número de caracteres de x.
Seguro nunca se preguntaron de dónde venía ese método, solo lo usaron. Viene del prototipo de String que está declarado en el core de Javascript.

Esto nos lleva a lo siguiente y es que un objeto puede tener como parámetro, una función.

				
	var x = {
			x: "hola",
			y: "mundo",
			w: {
				x: "wow",
				y: "Inception"
			},
			miFuncion: function(a,b,c) {
				return a-b+c;
			}
		};

	// Y la llamamos así:

	x.miFuncion(3,2,5);

	// También podemos agregar funciones a objetos así:

	x.nuevaFuncion = function() {
		console.log("Aquí se hace algo");
	}

Los objetos de javascript son maravillosos, flexibles, orgánicos y simples de entender y manejar. Además, es de vital importancia entenderlos para poder usar cualquier tipo de librería o api que existe.

JSON

JSON significa "JavaScript Object Notation" y, como su nombre lo indica, se refiere a la manera en que se definen los objetos de javascript. JSON es independiente del lenguaje, es decir, es una forma universal de transmitir datos entre distintos tipos de aplicaciones. No es nada pesado, por lo que es la mejor manera de pedir y enviar datos a una BD. Finalmente, JSON es auto descriptivo, lo que lo hace fácil de entender.

El siguiente es un ejemplo de un documento de JSON válido:

				
	{
	"estudiantes":[
	    {"primerNombre":"Diego", "apellido":"Barbosa"}, 
	    {"primerNombre":"Juan Carlos", "apellido":"Peláez"},
	    {"primerNombre":"Juan David", "apellido":"Pérez"}
	]
	}

Como ven, la notación es muy similar a la de un objeto de Javascript con unas cuantas diferencias. Estas son las reglas de sintaxis de JSON:

  • La información va en pares de nombre/valor.
  • Los nombres y valores ambos van entre comillas.
  • Los corchetes contienen objetos.
  • Las llaves contienen arreglos.
  • El documento de JSON total siempre va entre corchetes (es un objeto).

Finalmente, para poder usar JSON, se usan los siguientes métodos propios de Javascript:

				
	var json = '{
	"estudiantes":[
	    {"primerNombre":"Diego", "apellido":"Barbosa"}, 
	    {"primerNombre":"Juan Carlos", "apellido":"Peláez"},
	    {"primerNombre":"Juan David", "apellido":"Pérez"}
	]
	}'

	// Para decodificar:

	var decode = JSON.parse(json);

	// Para encodificar un objeto:

	var foo = {x:"bar", y:1};

	var encode = JSON.stringify(foo);
