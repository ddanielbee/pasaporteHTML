Ejercicio Teórico

En Javascript, entender la teoría es importante. Muchas veces nos vamos a encontrar con código que no escribimos nosotros, mal documentado y necesitamos entender qué dice ahí para poder usarlo.

Para este ejercicio, volvamos al sitio web que vimos anteriormente Javascript.com. Una vez ahí, tomemos el test que nos dicen al final.

La dificultad de esas preguntas puede ser alta, entonces no se desanimen si no saben todas las respuestas.
