ToDo App - Segunda Parte

Volviendo el esqueleto una aplicación funcional.

En este paso vamos a incluirle interacción a nuestra aplicación por medio de Angular.

Lo que vamos a hacer

Este es un listado de las cosas que nuestra aplicación debe poder hacer al terminar esta parte del curso:

  • 1. Debe llamar los diferentes ToDo's del modelo de información.
  • 2. Deben poder agregarse otros ToDo's al modelo de información.
  • 3. Deben poderse marcar como terminados nuestros ToDo's.
  • 4. Los ToDo's terminados deben desaparecer de la vista, a menos que la casilla para verlos esté marcada.
  • 5. Debemos poder filtrar nuestro ToDo's por su nombre.

Los elementos de Angular que vamos a utilizar para lograr esto serán:

  • ng-repeat
  • ng-click
  • ng-model
  • ng-hide
  • $scope
  • filters
  • Views, Module, Controllers
Todo lo que empieza por ng- es una directiva, es decir, se utiliza directamente en el html.
